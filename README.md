- MongoDB

## Installation:

`Yarn`
Installer les dépendances avec la commande `yarn add`

Démarrer le serveur avec la commande `yarn start`

Ou
`Npm`
Installer les dépendances avec la commande `npm install`

Démarrer le serveur avec la commande `npm start`
